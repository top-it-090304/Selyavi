# 🛡️ Tanks 2026

![Godot Engine](https://img.shields.io/badge/Engine-Godot%204.x-blue?logo=godot-engine&logoColor=white)
![Platform](https://img.shields.io/badge/Platform-Android%20%7C%20Windows-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

**Tanks 2026** — это динамичный аркадный танковый экшен, вдохновленный классикой, но перенесенный в сеттинг недалекого будущего. Сражайся, улучшай свой танк и уничтожай базы противника!

---

## 🚀 Основные особенности

- **⚔️ Эпические сражения:** Сражайся с умными противниками на разнообразных аренах.
- **🛠️ Система вооружения:** Три типа снарядов (Плазма, Средние и Легкие снаряды) с уникальными характеристиками и временем перезарядки.
- **📱 Мобильный геймплей:** Полностью адаптированное управление под Android (джойстики, тач-интерфейс, оптимизированный выбор уровней).
- **📊 Прогрессия:** Система уровней (миссий) с нарастающей сложностью и боссами на каждом 5-м этапе.
- **💾 Сохранения:** Твой прогресс и заработанные деньги всегда под защитой.

---

## 🎮 Управление

### Android
- **Левый джойстик:** Движение танка.
- **Правый джойстик (Aim):** Поворот башни и прицеливание.
- **Тап по иконкам снарядов:** Смена типа боеприпасов.

### Windows (Desktop)
- **WASD:** Движение.
- **Mouse:** Прицеливание и стрельба.
- **Цифры 1-3:** Смена снарядов.

---

## 🛠️ Технологии

Проект разработан на **Godot Engine 4**. 
В игре реализованы:
- Плавная система прокрутки уровней с поддержкой тач-жестов.
- Динамическая система HUD (здоровье, жизни, деньги, боезапас).
- Гибкая архитектура уровней на базе `TileMapLayer`.

## 📥 Установка и запуск

1. Установи [Godot Engine 4.x](https://godotengine.org/).
2. Клонируй репозиторий:
   ```bash
   git clone https://github.com/your-username/tanks2026.git
   ```
3. Открой `project.godot` в редакторе.
4. Для Android: Настрой экспорт и собери APK (в корне уже есть готовый пример `Tanks 2026.apk`).
