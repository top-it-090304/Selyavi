## Особенности

- **Боевая система** — сражайся с врагами на разнообразных аренах с укрытиями и препятствиями
- **Три слота для снарядов** — настраивай боезапас перед каждой миссией: рикошет, БОПС, фугас, плазма и другие
- **Магазин с апгрейдами** — три ветки прокачки: **Танк** (пушка, корпус), **Штаб** (инновации: автономная турель), **Снаряды**
- **Прогрессия миссий** — нарастающая сложность, каждый 5-й этап — **бой с боссом**
- **Разные биомы** — летние поля, зимние карты, городские арены
- **Мобильный геймплей** — двойной джойстик, тач-интерфейс, оптимизация под Aurora OS
- **Система сохранений** — прогресс и валюта сохраняются между сессиями

## Технологии

| | |
|--|--|
| **Движок** | [Godot Engine 4.4](https://godotengine.org/) |
| **Язык** | GDScript |
| **Платформа** | Aurora OS (armv7hl) |
| **Форматы текстур** | ETC2, ASTC |
| **Архитектура** | TileMapLayer, AutoLoad-синглтоны, Resource-система данных |

### Структура проекта

```
Tanks_2026/
├── scenes/          # Сцены: Field, Menu, Tank, Traps, Walls...
├── scripts/         # GDScript: логика игры, менеджеры, ресурсы
│   ├── AudioManager.gd
│   ├── GameManager.gd
│   ├── SaveManager.gd
│   └── LoadingManager.gd
├── assets/          # Спрайты, музыка, тайлсеты
├── resources/       # .tres ресурсы данных (TrapData, BulletData...)
└── Images/          # Скриншоты
```
